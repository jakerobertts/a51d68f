Common Variables

  • The independent variable is the one that is changed or controlled in an experiment to test its effects on the dependent variable.
  • The dependent variable is the one that is measured in an experiment and is affected by the independent variable.
  • The control variable is kept constant to ensure that the results are due to the independent variable only.
  • The confounding variable is an outside influence that can affect the dependent variable, making it difficult to determine the true relationship between the independent and dependent variables.
  • The extraneous variable is any variable other than the independent variable that could influence the dependent variable, but is not of primary interest in the study.
  • The moderator variable is a variable that affects the strength or direction of the relationship between the independent and dependent variables.
  • The mediator variable is a variable that explains the mechanism through which the independent variable affects the dependent variable.
  • The operational definition is a clear, precise, and measurable definition of a variable that allows it to be tested and measured in an experiment.
  • The null hypothesis is a statement that there is no effect or no difference, and it is tested against the alternative hypothesis, which states that there is an effect or a difference.
  • The alternative hypothesis is a statement that there is an effect or a difference, and it is tested against the null hypothesis, which states that there is no effect or no difference.
  • The sample size is the number of observations or data points collected in a study, which affects the reliability and validity of the results.
  • The significance level (alphA. is the threshold for determining whether a result is statistically significant, typically set at 0.05 or 0.01, indicating the probability of rejecting the null hypothesis when it is true.
  • The p-value is the probability of obtaining results at least as extreme as those observed, assuming the null hypothesis is true, and it is used to determine statistical significance.
  • The confidence interval is a range of values that is likely to contain the true population parameter with a specified level of confidence, often 95% or 99%.

Equations

  • $$W = F \cdot d \cdot \cos \theta$$ Find the work done by a force on an object over a distance at an angle. The angle is nonzero when the force is not parallel to the object's movement.
  • $$KE = \frac{1}{2} mv^2$$ Find the kinetic energy using the object's mass and current velocity.
  • $$PE = mgh$$ Find the potential energy using the object's mass and height.
  • $$W=\Delta KE$$ The work done on an object is equal to the change in its kinetic energy.
  • $$P=\frac{W}{t}$$ Find the power using the work done and time.
  • $$F = ma$$ Find the force using the object's mass and acceleration.
  • $$F = \frac{Gm_1m_2}{r^2}$$ Find the gravitational force between two masses.

  • $$v = V_0 + at$$ Find the final velocity using the initial velocity, acceleration, and time.
  • $$x = V_0 t + \frac{1}{2} a t^2$$ Find the displacement using the initial velocity, acceleration, and time.
  • $$v^2 = V_0^2 + 2a(d)$$ Find the final velocity using the initial velocity, acceleration, and displacement.
  • $$a_c=\frac{v^2}{r}$$ Find the centripetal acceleration using the velocity and radius.

  • $$F_c = \frac{mv^2}{r}$$ Find the centripetal force using the mass, velocity, and radius.
  • $$\tau = rF \cdot \sin \theta$$ Find the torque using the radius, force, and angle.
  • $$P=\frac{F}{A}$$ Find the pressure using the force and area.
  • $$\rho = \frac{m}{V}$$ Find the density using the mass and volume.
  • $$P=pgh$$ Find the pressure at a depth using the fluid density, gravitational acceleration, and height.
  • $$F_b=\rho Vg$$ Find the buoyant force using the fluid density, object volume, and gravitational acceleration.

  • $$A_1V_1=A_2V_2$$ Find the flow rate using the cross-sectional areas and velocities.

  • $$P + \frac{1}{2}\rho v^2 = \text{constant}$$ Find the pressure in a fluid flow using the fluid density and velocity.
  • $$\Delta U = Q - W$$ Find the change in internal energy using the heat added and work done.
  • $$Q = mc \Delta T$$ Find the heat transfer using the mass, specific heat capacity, and temperature change.
  • $$v = \lambda f$$ Find the wave speed using the wavelength and frequency.

  • $$f = \frac{1}{T}$$ Find the frequency using the period.
  • $$E = hf$$ Find the photon energy of a photon using Planck's constant and frequency.

  • $$E = \frac{hc}{\lambda}$$ Find the photon energy of a photon using Planck's constant and wavelength.
  • $$n_1 \sin \theta_1 = n_2 \sin \theta_2$$ Find the relationship between the angles and indices of refraction for two media.
  • $$\frac{1}{o} + \frac{1}{i} = \frac{1}{f}$$ The lens/mirror equation relating object distance, image distance, and focal length.
  • $$V = IR$$ Find the voltage using the current and resistance.
  • $$P = IV$$ Find the power using the current and voltage.
  • $$P = I^2R$$ Find the power using the current and resistance.
  • $$P = \frac{V^2}{R}$$ Find the power using the voltage and resistance.
  • $$F_e = \frac{kq_1q_2}{r^2}$$ Find the electrostatic force between two charges.
  • $$E = \frac{kq}{r^2}$$ Find the electric field due to a point charge.
  • $$F = qE$$ Find the force on a charge in an electric field.
  • $$F = qvB\sin\theta$$ Find the magnetic force on a charge moving in a magnetic field.
  • $$n = \frac{mass}{Mw}$$ Find the number of moles using the mass and molar mass.
  • $$M = \frac{n}{V}$$ Find the molarity using the number of moles and volume.
  • $$M_1V_1 = M_2V_2$$ The dilution equation relating concentrations and volumes.
  • $$PV = nRT$$ The ideal gas law relating pressure, volume, and temperature.
  • $$P = \sigma P_i$$ Find the pressure at a depth in a fluid using the fluid density and gravitational acceleration.
  • $$\frac{V_1}{V_2} = \sqrt{\frac{M_2}{M_1}}$$ Find the relationship between the volumes and molarities before and after dilution.
  • $$pH = -\log[H^+]$$ Find the pH of a solution using the hydrogen ion concentration.
  • $$pOH = -\log[OH^-]$$ Find the pOH of a solution using the hydroxide ion concentration.
  • $$pH + pOH = 14$$ Find the relationship between pH and pOH in a solution.
  • $$K_a = \frac{[H^+][A^-]}{[HA]}$$ Find the acid dissociation constant using the concentrations of the products and reactants.
  • $$K_b = \frac{[OH^-][BH^+]}{[B]}$$ Find the base dissociation constant using the concentrations of the products and reactants.
  • $$pH = pK_a + \log\left(\frac{[A^-]}{[HA]}\right)$$ Find the pH of a solution using the acid dissociation constant and concentrations.
  • $$K_{eq} = \frac{[C]^c[D]^d}{[A]^a[B]^b}$$ Find the equilibrium constant using the concentrations of the products and reactants.
  • $$Q = \frac{[C]^c[D]^d}{[A]^a[B]^b}$$ Find the reaction quotient using the concentrations of the products and reactants.
  • $$\Delta G = \Delta G° + RT\ln Q$$ Find the change in Gibbs free energy using the standard change and reaction quotient.
  • $$G = H - TS$$ Find the Gibbs free energy using the enthalpy, temperature, and entropy.
  • $$\Delta G° = -RT\ln K_{eq}$$ Find the standard change in Gibbs free energy using the equilibrium constant.
  • $$Rate = k[A]^a[B]^b$$ Find the rate of a reaction using the rate constant and concentrations.
  • $$k = Ae^{-\frac{E_a}{RT}}$$ Find the rate constant using the Arrhenius equation.
  • $$\ln\left(\frac{k_2}{k_1}\right) = \frac{E_a}{R}\left(\frac{1}{T_1} - \frac{1}{T_2}\right)$$ Find the relationship between the rate constants and temperatures before and after a reaction.
  • $$G = -nFE$$ Find the Gibbs free energy using the number of moles, Faraday's constant, and cell potential.
  • $$G = -nF_{cell}$$ Find the Gibbs free energy using the number of moles and cell potential.
  • $$Q_{elec} = nF$$ Find the electric charge using the number of moles and Faraday's constant.
